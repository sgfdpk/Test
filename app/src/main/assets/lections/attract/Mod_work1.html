Атракції - система розваг і заходів, і спрямованих на те, щоб сформувати позитивне враження у туриста від туру

Класифікація системи атракцій:

За масштабом:
• туристського регіону
• країни
• адміністративно-територіальної одиниці країни
• туристичної (історичної) частини міста / села / смт

За метою відвідування:

• пізнавально-історична

• рекреаційно-оздоровча
• ділова (службова, бізнес, навчання)
• спеціалізована (релігійна, спортивна та ін.)

За ступенем навантаження (частотою відвідування):
• слабко навантажена
• навантажена
• перенавантажена

За розвитком інфраструктури:

• розвинуті навколо центра атракції місця для відвідувачів з особливими потребами, місця для паркування, дороги, торговельні точки
• відсутні місця для відвідувачів з особливими потребами, місця для паркування, дороги, торговельні точ

• частково присутні / в поганому стані навколо центра атракції місця для відвідувачів з особливими потребами, місця для паркування, дороги, торговельні точки
